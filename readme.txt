== Description ==

Install plugin and create your simple alert popup box.
Chose one or more products edit it, go to "product details" ->  "Product popup" and check "Add product to marketing popup"
You can using a WordPress editor to write before the product.
There are different option you can chose:

* One or more product highlights
* Show popup at the first visit of the day 
* Contact Form 7 ready
* Shortcode ready (you can use any shortcode)
* Full control on popup content
* Standard WordPress content editor
* Set a delay to start popup box
* Responsive mobile friendly popup
* Insert image in popup box
* Test mode (NEW FEATURES!)

For more information about the PREMIUM version of NS Product Marketing Popup for WooCommerce, visit its official page on <a href="http://www.nsthemes.com/">NsThemes marketplace</a>.

== Installation ==

1. Upload the entire 'ns-product-marketing-popup-for-woocommerce' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. View admin menu, then click on 'Settings' -> 'NS Product Marketing Popup for WooCommerce'
